# 移动端适配方法
1.  百分比(流式布局)  
    **对一个元素的比例没有特殊要求的时候,可以用百分比来进行布局**  
2.  响应式  
    **工作大，维护性难**
3.  viewport  
    **通过动态的设置窗口比例,使得我们可以使用原图大小来进行布局**  
    **这个方法简单粗暴，又高效,但是使用过程中反应缩放会导致有些页面元素会模糊**
4.  弹性盒模型  
    **适合对一行有多个内容或有多行内容来进行批量处理的情况**
   
5.  rem  
    **rem（font size of the root element）是指相对于根元素的字体大小的单位**
